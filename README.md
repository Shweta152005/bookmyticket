# BookMyTicket

An online ticket booking system developed using Java, Spring Boot, Servlets, JSP, and MySQL.

## Features

- User Registration and Login
- Ticket Search and Booking
- Booking Management
- Database Integration
- Responsive User Interface

## Tech Stack

- Java
- Spring Boot
- Servlets
- JSP
- MySQL
- HTML
- CSS

## Project Structure

- Frontend: JSP, HTML, CSS
- Backend: Java, Spring Boot, Servlets
- Database: MySQL

## Learning Outcomes

- Implemented MVC architecture
- Applied Object-Oriented Programming concepts
- Integrated database operations using MySQL
- Developed web-based booking functionalities
